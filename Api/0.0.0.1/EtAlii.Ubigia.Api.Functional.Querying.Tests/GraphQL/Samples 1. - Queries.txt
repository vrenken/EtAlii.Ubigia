### 01 Query with nodes directive at entity level and normal property
query data
{ 
    person @nodes(path:"person:Stark/Tony") 
    { 
        nickname
    } 
}

### 02 Query with nodes directive at entity path and relative property
query data 
{ 
    person @nodes(path:"person:Stark/Tony")
    { 
        lastname @id(path:"\\") 
    } 
}

### 03 Query with nodes directives at multiple entity paths unioned together returning multiple entries and a normal property
query data 
{ 
    person 
    @nodes(path:"person:Vrenken/Peter", mode: "Union")
    @nodes(path:"person:Vrenken/Tanja")
    { 
        nickname 
    } 
}

### 04 Query with multiple entity paths unioned together returning multiple entries and a normal property
query data 
{ 
    person 
    @nodes(path:"person:Vrenken/Peter", mode: "Union")
    @nodes(path:"person:Vrenken/Tanja")
    { 
        name @id 
    } 
}

### 06 Unnamed query.

{ 
    location @nodes(path:"location:DE/Berlin//", mode: "Intersect")
    time @nodes(path:"time:2012//")
    person @nodes(path:"person:Vrenken/Peter//")
    { 
        firstname 
        lastname @id(path:"\\") 
    }
}
